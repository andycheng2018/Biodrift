## ⌨️ Controls

| Action     | Key             |
|------------|------------------|
| Move       | WASD / Arrows    |
| Shoot      | Space            |
| Dash       | Shift            |
| Pause/Menu | Esc              |

> You can rebind keys in `settings.json`.
